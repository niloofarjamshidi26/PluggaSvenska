# PluggaSvenska

PluggaSvenska 🇸🇪
A lightweight Swedish grammar study tool built for SFI (C3A level) learners.
→ Open the App

What it does
A mobile-friendly checklist covering the core grammar topics needed for SFI progression:

Verb tenses (presens, preteritum, perfekt, futurum)
Bestämd / obestämd form (singular & plural)
Plural groups (1–5)
Adjective agreement and comparison
Numbers, clock, and dates

Each topic breaks down into specific sub-topics with examples. Check off what you've studied and track your overall progress.
Why I built it
I'm learning Swedish while job searching in Stockholm. I needed a simple tool to track what I'd actually reviewed — not just what I'd passively read. Nothing I found matched exactly what I needed at SFI C3A level, so I built it.
Tech
Single-file HTML/CSS/JS — no framework, no build step, no backend. Hosted on GitHub Pages. Progress is saved in localStorage.
